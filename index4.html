Crypto Wallet Dashboard This project is a responsive web dashboard that showcases a simple crypto wallet interface using HTML and CSS. It features a clean and modern UI displaying the total wallet balance and a list of tracked cryptocurrencies like Bitcoin, Ethereum, and Dogecoin. Each coin includes a price display and a “Set Alert” button for potential future interactions. The layout is lightweight, mobile-friendly, and built with a minimalist design, making it an excellent starter template for anyone learning front-end development or experimenting with crypto-related projects.
